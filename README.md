# MEEG-MNE-BIDS [3]: Source Analysis 

This Brainlife App runs the sensor-level analysis of MNE-BIDS-pipeline on the derivatives produced by the previous preprocessing App (MNE-BIDS-Pipeline [2]: Sensor Analysis).
It estimates cortical sources of M/EEG activity for EEG and MEG data, covering the following steps:
1. Takes the sensor-level BIDS derivatives generated by Sensor analysis App as input.
2. Builds the head model nedeed for source estimation:
   1. If the specific T1w of a subject is provided and no template MRI is selected, the App runs FreeSurfer's recon-all to reconstruct the subject's anatomy (it requires a FreeSurfer license).
   2. Alternatively, a template MRI can be used to skip recon-all.
   3. The Boundary Element Model (BEM) surfaces are then computed.
3. Builds the source space and the forward solution.
4. Computes the noise covariance and the inverse operator (configurable method) and applies it to the data to obtain source estimation.
5. HTML reports summarizing the source-level analyses results.

Note that source estimation is skipped if the recording is resting-state and no evoked data was defined at the sensor analysis step.

# Input 

- The BIDS derivatives folder produced by the M/EEG Sensor Analysis App (bids_dir) with a config.json generated by Brainlife that sets the pipeline parameters.
- The subject's T1-Weighted MRI (t1) can be provided as an optional input to run FreeSurfer's recon-all and build a specific head model. If omitted the source estimation can be computed on a template anatomy by setting the use_template_mri parameter.

# Output 

- out_dir: including a copy of the inputs BIDS derivatives plus the soruce-level outputs.
- html_report

# How to run this App on Brainlife 

The App runs inside a Singularity container built from ghcr.io/carlota-juarez/app-mne-bids:full, based on python:3.11-slim with mne, mne-bids, mne-bids-pipeline, and FreeSurfer installed.
Because recon-all and source modeling are computationally demanding, this App requests more memory and walltime than the Preprocessing and Sensor Analysis Apps. These are the steps to run the App:

1. Register on Brainlife.io and create a project.
2. Select this App from the App page. Set the output of the previous App as the input and configure the provided parameters to suit the selected data.
3. Execute the App. Once completed, the results can be dwonloaded.
